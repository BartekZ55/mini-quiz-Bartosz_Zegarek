Mini Quiz – Bartosz Zegarek
 
Opis projektu
Aplikacja mobilna na Androida, która pozwala użytkownikowi rozwiązać prosty quiz o stolicach wielokrotnego wyboru.  
Użytkownik wybiera odpowiedzi na losowe pytania, a aplikacja zlicza poprawne odpowiedzi i wyświetla wynik.  
Dostępna jest też opcja resetowania quizu.
 
Instrukcja uruchomienia
1. Otwórz projekt w Android Studio  
2. Upewnij się, że masz włączony emulator lub podłączone urządzenie Android  
3. Kliknij run (zielona strzałka)
4. Na ekranie głównym kliknij ROZPOCZNIJ QUIZ
5. Odpowiadaj na pytania i obserwuj wynik  
6. Aby zacząć od nowa, kliknij RESETUJ QUIZ
